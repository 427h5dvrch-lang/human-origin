# Guide d'installation — HumanOrigin (macOS)

## Ce dont tu as besoin

- Un Mac sous macOS 12 (Monterey) ou version ultérieure.
- Le fichier `.dmg` fourni (lien communiqué séparément).
- Environ 5 minutes.

---

## Étape 1 — Télécharger le DMG

Télécharge le fichier `HumanOrigin_X.X.X.dmg` depuis le lien partagé.  
Si ton navigateur le place dans `Téléchargements`, ouvre ce dossier.

---

## Étape 2 — Ouvrir le DMG

Double-clique sur le fichier `.dmg`.  
Une fenêtre s'ouvre avec l'icône de l'application.

---

## Étape 3 — Installer l'application

Glisse l'icône `HumanOrigin` vers le dossier `Applications` dans la même fenêtre.  
Si la fenêtre ne montre pas ce raccourci, ouvre `Finder → Applications` et glisse l'icône manuellement.

---

## Étape 4 — Premier lancement

Ouvre le dossier `Applications` et double-clique sur `HumanOrigin`.

**Si macOS affiche une alerte de sécurité** ("Apple ne peut pas vérifier que cette app...") :

1. Ouvre `Réglages système → Confidentialité et sécurité`.
2. En bas de la section "Sécurité", tu verras un message mentionnant HumanOrigin.
3. Clique sur **Ouvrir quand même**.
4. Relance l'application.

> L'application est signée avec un identifiant développeur Apple et notarisée par Apple. L'alerte initiale est un comportement standard de macOS pour les apps distribuées hors App Store.

---

## Étape 5 — Accepter les permissions

Au premier lancement, macOS demande deux autorisations nécessaires au fonctionnement de l'outil :

**Accessibilité (Contrôle de l'ordinateur)**  
Permet à HumanOrigin d'enregistrer l'activité clavier et souris.

**Surveillance des entrées (Input Monitoring)**  
Permet d'observer les frappes clavier pendant une session.

Pour chaque permission :
1. Clique sur **Ouvrir les réglages** dans la fenêtre d'alerte.
2. Dans `Réglages système → Confidentialité et sécurité`, active HumanOrigin dans la liste correspondante.
3. Reviens dans l'application.

> Ces permissions restent locales à ta machine. Aucune frappe n'est transmise à un serveur.

---

## Vérification

L'application est correctement installée si :

- L'interface principale s'affiche sans alerte persistante.
- Le bouton de démarrage d'observation est accessible.

En cas de problème, contacte-moi directement.
