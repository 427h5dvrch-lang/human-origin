# HumanOrigin Cartouche — Physical Dimensions Standard

## Reference document format
Default reference: A4 document.

## Cartouche size
- Width: 60 mm
- Height: 22 mm

## QR code
- QR size: 16 mm x 16 mm
- Must remain scannable when printed
- Minimum print quality: 300 dpi recommended

## Margins and placement
Default placement: bottom-right of the first page.

Margins from page edges:
- Right margin: 12 mm
- Bottom margin: 12 mm

This placement ensures:
- visibility,
- non-intrusion,
- compatibility with most document layouts,
- consistency across documents.

## Internal layout
Recommended internal structure:

| Text block | QR block |
|------------|----------|
| HumanOrigin | QR |
| Document-bound proof | |
| ID | |
| Verify record | |
| Doc (optional) | |

## Typography (print)
Recommended print sizes:
- "HumanOrigin": ~9 pt
- "Document-bound proof": ~7 pt
- ID / Verify / Doc: 6.5–7 pt

Font must remain readable when printed.

## Color
Recommended:
- Black or dark grey text
- White background
- One blue accent allowed
- High contrast for print

Avoid:
- gradients
- shadows
- glossy effects
- multiple colors

## Multipage documents
For multipage PDFs:
- First page: full cartouche (60 x 22 mm)
- Other pages: compact version allowed
  - Width: 45 mm
  - Height: 18 mm
  - QR still present

## Strategic objective
The cartouche must:
- remain readable when printed,
- remain discreet,
- remain consistent across documents,
- become recognizable as a standard mark.

Consistency is more important than decoration.
